### J - [[Jump Instruction]]

| ***Section*** | opcode | target |
| ------------- | ------ | ------ |
| ***Size*** | 6 bits | 26 bits |
| `j label (123)` | 000010 | 00 00000000 00000000 01111011 |

#### Sections
- opcode - Describes what operation to do. Addition has one, jump one, or one etc.
- Target - [[Address]] to jump to
